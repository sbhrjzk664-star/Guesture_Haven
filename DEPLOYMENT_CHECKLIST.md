# Pre-Deployment Checklist

Complete this checklist before connecting your domain and going live:

## Content & Customization
- [ ] Update company name/branding throughout
- [ ] Replace placeholder email addresses with real contacts
- [ ] Update WhatsApp business number (0506993828 → your number)
- [ ] Add real product images/descriptions
- [ ] Update company address and contact info
- [ ] Review all text for typos and accuracy
- [ ] Add social media links if applicable

## Technical Setup
- [ ] Test all links (internal and external)
- [ ] Verify WhatsApp links work on mobile
- [ ] Test shopping cart functionality
- [ ] Test form submissions (wholesale inquiry)
- [ ] Check responsive design on mobile (use Chrome DevTools)
- [ ] Test on different browsers (Chrome, Firefox, Safari, Edge)
- [ ] Verify images load correctly
- [ ] Check page load speed (Google PageSpeed Insights)

## SEO & Meta Tags
- [ ] Add Google Meta tags (index.html, store.html, wholesale.html)
- [ ] Add Open Graph tags for social sharing
- [ ] Create sitemap.xml
- [ ] Add robots.txt file
- [ ] Optimize page titles and descriptions
- [ ] Add schema markup (structured data)

## Security & Performance
- [ ] Enable SSL/HTTPS (automatic with Vercel/Netlify)
- [ ] Set up security headers
- [ ] Enable GZIP compression
- [ ] Minimize CSS/JS files (optional for static site)
- [ ] Add favicon.ico
- [ ] Test contact forms for spam

## Analytics & Monitoring
- [ ] Set up Google Analytics
- [ ] Add Google Search Console
- [ ] Set up error monitoring (optional)
- [ ] Configure email notifications

## Domain & Hosting
- [ ] Register domain name
- [ ] Choose hosting platform (Vercel/Netlify recommended)
- [ ] Deploy to hosting
- [ ] Point domain DNS
- [ ] Wait 24-48 hours for DNS propagation
- [ ] Test site on custom domain

## Final Checks Before Launch
- [ ] WhatsApp numbers work correctly
- [ ] All pages accessible from navigation
- [ ] Mobile menu (if applicable) works
- [ ] Footer links work
- [ ] Form sends messages to WhatsApp
- [ ] No console errors (check DevTools)
- [ ] Lighthouse score 90+
- [ ] All pages load under 3 seconds

## Post-Launch
- [ ] Monitor error logs
- [ ] Check analytics traffic
- [ ] Gather customer feedback
- [ ] Respond to inquiries promptly
- [ ] Monitor uptime
- [ ] Regular backups
